Summary: Latitude vs. X


The purpose of this project was to compare and analyze weather trends based on distance from the equator. A random list of 628 cities were queried using OpenWeatherMap API to retrieve forecasts from that day, June 28, 2019. The list of cities itself was collected by passing a 1500x1500 grid of coordinate pairs through the citipy module, which locates nearby cities.

The dataset was then plotted using Matplotlib on four attributes vs latitude: max temp, humidity, cloudiness, and wind speed. The results of the study are further detailed by navigating the menu above, along with analysis and graphs.
